在```TypeScript```中存在两种声明空间：
1. 类型声明空间
2. 变量声明空间


### 类型声明空间 
```ts
class Foo {}
interface Bar {}
type Bas = {}
```
你可以将他们作为类型注解使用
```ts
let foo: FOO
let bar: Bar
let bas: Bas
```
尽管你定义```interface Bar```,但并不能作为一个变量使用。因为它没有定义在变量声明空间中。
```ts
interface Bar {}
const bar = Bar // Error: "cannot find name 'Bar'"
```

### 变量声明空间
变量声明空间包含可用作变量的内容，  
在上文中```class Foo```提供了一个类型```Foo```到类型声明空间，此外。  
它同样提供了一个变量```Foo```到变量声明空间。
```ts
class Foo {}
const foo = Foo
```
